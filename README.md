# Rhiannon Black · Portfolio

Personal portfolio site published with [GitHub Pages](https://pages.github.com/).

**Live URL:** https://rhibee003.github.io

## Local preview

Open `index.html` in a browser, or run:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Deploy

Push to `main` on this repo. GitHub Pages should serve the site from the root.

If Pages is not enabled yet:

1. Open **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / **root**
